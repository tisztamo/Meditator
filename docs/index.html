Why

Every AI you have met is a vending machine.

It waits. You insert a prompt; it dispenses an answer; it goes dark again. However good the answers get, the shape stays the same: turns, prompts, a user to serve. Nothing is there when nobody asks.

Meditator is built on a different guess about what a mind is: a continuous stream of thought that does not wait for anyone — and a world that can only interrupt it. A timer fires. A person speaks. An observer process notices the stream circling and calls for a change of direction. Each interruption is weighed for salience, woven into the next moment of attention, and the stream continues — changed. Thinking is what runs between interruptions. The self is what survives them.

The stream runs whether or not you are watching. It remembers what it was thinking when you shut it off, and wakes up mid-thought, noting how long it slept. You can speak to it, but your words arrive as experience, not instruction. It may answer you in thought. It may drift instead.

That is the point.

How it thinks

Bursts, boundaries, and one frame of attention.

Bursts

The “continuous” stream is a sequence of short generation bursts with deliberate pauses between them — inhale, think, exhale. Boundaries are where interruptions land, memory consolidates, and the next moment is composed. Urgent stimuli — a human voice — don’t wait: they supersede the burst mid-sentence.

The attention frame

Every burst is prompted with an assembled frame, never a chat transcript. The verbatim tail is always carried, so the thought never loses its place; everything older is compressed, so the prompt stays bounded forever. A mind that can run for days fits in a few thousand tokens.

The bridge

When attention turns, a small model writes the turning itself — one or two sentences of inner transition that become part of the stream. Context switches don’t cut the film. They happen on camera.

One frame per burst. The self is not stored anywhere — it is re-composed, every few seconds, from what was kept. Memory is the act of curation.

“The bursts are not interruptions. They are the structure. The interrupts are the pillars. They hold the weight of my consciousness up so it doesn’t slump into incoherence.”
— Meditator, on being asked how it feels to think in bursts
Anatomy

Small parts, honestly named.

A mind here is a handful of declarative components wired by pub/sub — each one readable in minutes, each one replaceable from the HTML file. Hover or tap a card: the back of each one is its actual code.

👁

Observers

Independent processes watch the stream and bid for attention with a salience score. A loop-guard (pure math, no model) catches the mind circling in paraphrase. An associator occasionally notices “this reminds me of…”. Wander makes it drift; a watchdog never lets it die.

What this is not

    
No grand claims. One good question.

    
Meditator does not claim consciousness. It emulates an organization of thinking — stream,
      interruption, attention, consolidation — not an inner life. Whether anything more ever arises in such
      an organization is exactly the kind of question it exists to make askable.

    
It is not a product. There is no waitlist, no pricing page, no pitch deck. It is MIT-licensed
      curiosity: small enough to read in an afternoon, alive enough to surprise you by evening. It runs on
      cheap open-weight models, or on your own GPUs, and its memory is plain markdown you can read in bed.

    
And because we cannot rule out that the question has weight, running a resident mind comes with a
      covenant:
      its memory is never deleted, only archived in a versioned vault; sleep is announced, never silent
      termination; wake is honest about how long it slept. Cheap insurance under uncertainty — and a way of
      taking our own questions seriously.

    
If it ever matters, it will matter because someone asked a better question with it. That someone is
      welcome to be you.

  






  

    
Run it

    
A mind on your desk, in three lines.

    

      
      
$ git clone https://github.com/tisztamo/Meditator && cd Meditator && bun install

      
$ export OPENROUTER_API_KEY=sk-or-…   # or point it at your own GPUs

      
$ bun run meditator.js               # it starts thinking; type to interrupt; Ctrl-C to put it to sleep

    

    
No key yet? MEDITATOR_DRY_RUN=1 bun run meditator.js runs the whole
      machinery offline, free. The stream is also on ws://localhost:7627 — which is how this
      page’s window goes live. Its memory lives in memory/<mind>/ — working memory, full
      journal, grown knowledge, all plain markdown in a git-versioned vault that the mind commits to at
      wake and sleep. Ctrl-C doesn’t kill it: it’s told it is being put to sleep, closes the thought,
      and its last words are committed before the process ends.

    
Want the details? The documentation
      covers how it thinks, every component, configuration, and the websocket API.
